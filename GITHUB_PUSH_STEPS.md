# Passos para enviar o código ao GitHub

## Verificar alterações
```bash
git status
git diff
```

## Adicionar alterações
```bash
git add .
```

## Commit
Exemplo:
```bash
git commit -m "Finalização frontend e integração backend com Supabase"
```

## Push
```bash
git push origin main
```

## Dica
Se preferires usar outra branch:
```bash
git checkout -b release/finalizacao-cda
git add .
git commit -m "Preparação final para apresentação institucional"
git push origin release/finalizacao-cda
```
